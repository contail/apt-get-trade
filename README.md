# apt-get-trade

> `apt-get install apartment`을 현실로.  
> 자동매매로 서울숲 트리마제 입성하는 그날까지.

---

## 📈 About
Python 기반 자동 주식 매매 프로그램  
한국투자증권 API 연동을 통해 국내 주식 실시간 매매 지원

---

## 🛠 Features
- 한국투자증권 OpenAPI 연동 (REST / WebSocket)
- 자동 매매 전략 실행
- 종목 모니터링 및 알림
- 거래 로그 저장
- 백테스트 기능 (추가 예정)

---

## ⚙️ Tech Stack
- Python 3.x
- FastAPI (백엔드)
- React (프론트, 예정)
- Docker (서버 배포)
- Redis / SQLite (캐시 및 DB)

---

- 서울숲 트리마제 입성
- 안정적인 자동 매매 수익률 달성
